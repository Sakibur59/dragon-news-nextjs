# 🐉 The Dragon News

> **"Journalism Without Fear or Favour"**

A modern, full-stack news web application built with **Next.js 14**, featuring category-based news browsing, secure authentication, and a clean responsive UI.

🔗 **Live Demo:** [dragon-news-nextjs-five.vercel.app](https://dragon-news-nextjs-five.vercel.app)

---

## ✨ Features

- 📰 **Category-based News** — Browse news by Breaking, Regular, International, Sports, Entertainment, Culture, Arts, and more
- 🔐 **Authentication** — Register & Login with Email, Google, and GitHub via Better Auth
- 👤 **User Profile** — Displays user name and avatar after login
- 📢 **Live News Ticker** — Scrolling latest news banner on the homepage
- 📱 **Fully Responsive** — Works seamlessly on all screen sizes
- ⚡ **Fast Performance** — Powered by Next.js App Router and server components
- ☁️ **Deployed on Vercel** — Fast global access with zero-config deployment

---

## 🛠️ Tech Stack

| Technology     | Purpose                              |
| -------------- | ------------------------------------ |
| Next.js 14     | Frontend + Backend (App Router)      |
| Tailwind CSS   | Styling & responsive design          |
| Better Auth    | Authentication & session management  |
| MongoDB        | Database for users & news data       |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Sakibur59/dragon-news-nextjs.git
cd dragon-news-nextjs
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 🌐 Deployment (Vercel)

1. Push your code to GitHub
2. Import the project on [vercel.com](https://vercel.com)
3. Add all environment variables in **Vercel Dashboard → Settings → Environment Variables**
   - Make sure to set `NEXT_PUBLIC_APP_URL` to your Vercel deployment URL
4. Click **Deploy**
